# 无人机通信“心跳”监测可视化系统

## 项目简介
本项目基于Python + Streamlit实现，模拟无人机每秒发送心跳包，地面站实时接收并检测掉线（3秒未收到即报警），同时可视化心跳包序号随时间的变化。

## 功能说明
1.  **模拟心跳包**：每秒自动发送含序号和时间的心跳包
2.  **掉线检测**：地面站实时判断，3秒未收到则触发超时报警
3.  **数据可视化**：用折线图展示心跳包序号随时间的变化
4.  **实时日志**：显示最近10条心跳包记录
